    列表 List¶

    定义

    • list = []

    操作

    • 增加

      • 列表.insert(索引, 数据) 在指定位置插入数据
      • 列表.append(数据) 在末尾追加数据
      • 列表.extend(列表2) 将列表2 的数据追加到列表

    • 修改

      • 列表[索引] = 数据 修改指定索引的数据

    • 删除

      • del 列表[索引] 删除指定索引的数据

        • 本质上是将一个变量从内存中删除， 后续代码将无法再使用这个变量

      • 列表.remove[数据] 删除第一个出现的指定数据

      • 列表.pop 删除末尾数据
      • 列表.pop(索引) 删除指定索引数据
      • 列表.clear 清空列表

    • 统计

      • len(列表) 列表长度
      • 列表.count(数据) 数据在列表中出现的次数

    • 排序

      • 列表.sort() 升序排序
      • 列表.sort(reverse=True) 降序排序
      • 列表.reverse() 逆序、反转

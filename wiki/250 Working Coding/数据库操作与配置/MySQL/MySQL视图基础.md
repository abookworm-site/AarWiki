# MySQL 视图基础

## 基本概念

- 视图是对若干张基本表的引用，一张虚表，查询语句执行的结果



## 试图操作

1， 定义视图

```sql
create view 视图名称 as select语句;
```

2， 查看视图

```sql
show tables;
```

3， 使用视图

```sql
select * from v_stu_score;
```

4， 删除视图

```sql
drop view 视图名称;
```



## 视图作用

1. 提高了重用性，就像一个函数
2. 对数据库重构，却不影响程序的运行
3. 提高了安全性能，可以对不同的用户
4. 让数据更加清晰

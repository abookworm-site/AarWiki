# C/C++ 编译运行环境 with VS Code

## 1. 安装VS code

下载并安装 **VS code** 软件。



## 2. Windows 安装 **MinGW-w64**

- 下载，解压*安装包*
- 将解压后的文件复制到 **C盘**，并复制该根目录中 **bin**所在的文件夹路径
- 将该路径添加到**系统环境 path**中
- 测试安装是否有效 `gcc -v`



## 3. 安装VS Code 插件：

- C/C++ - Visual Studio Marketplace
- Code Runner - Visual Studio Marketplace



## 4. 编译调试

- 编写C程序，运行即可。
